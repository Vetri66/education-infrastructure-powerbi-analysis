# Education Infrastructure Analysis in India (Power BI)

## Project Overview
This project analyzes education infrastructure in India using Power BI.  
The dashboard visualizes the number of schools across different education levels such as primary, upper primary, secondary, and senior secondary institutions.

## Dataset
Source: National Data & Analytics Platform (NDAP)

The dataset includes:
- State name
- Primary schools
- Upper primary schools
- Secondary schools
- Senior secondary schools

## Tools Used
- Power BI
- Excel / CSV
- NDAP Dataset

## Dashboard Features
- Total number of schools
- State-wise school distribution
- School type comparison
- Overall school distribution

## Key Insights
- Primary schools form the majority of institutions.
- Senior secondary schools are comparatively fewer.
- Some states have significantly higher education infrastructure.
